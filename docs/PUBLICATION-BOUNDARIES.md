# Publication Boundaries

## Public

- Sanitized summaries
- Original diagrams (Mermaid / SVG)
- General project facts
- Team names
- Supervisor name
- Institution and department
- Testing summary (including fails and inconclusive)
- Honest prototype status
- Lessons learned
- Modern proposals

## Private

- Raw academic PDF
- Student IDs
- Signature fields
- Raw extracted text
- Unapproved media
- Sensitive location data

## Deferred

- Redacted PDF
- Original hardware photographs
- GUI mockups
- Maps
- Report pages / screenshots
- Team role matrix
- Original source code
- Award supporting artifact

## Forbidden

- Fabricated completion
- Fake AI success
- Fake flight success
- Hidden failed tests
- Sole-builder claims
- Fake screenshots
