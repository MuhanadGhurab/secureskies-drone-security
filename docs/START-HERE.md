# Start Here

Welcome to the public SecureSkies portfolio repository.

## What this repository is

A sanitized, documentation-first historical portfolio of the 2024 SecureSkies
university graduation project, plus modern security and digital-twin planning.

## What this repository is not

- Not the private academic PDF
- Not a claim that the full autonomous system was deployed
- Not a real drone control or surveillance product
- Not the software digital twin (planned for milestone 4D)
