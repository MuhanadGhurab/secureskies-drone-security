- Physical perimeter risks and unauthorized access concerns

## Original objectives (stated)

- Combined physical + network security drone concept

- Hardware assembly and multiple integration checks
- Raspberry Pi OS tooling (Mission Planner, Wireshark/tcpdump)

- Architecture, requirements, and UI concept documentation
- Survey appendix validating perceived need (high-level)

- Regulatory / compliance readiness
- Conclusive ML real-time threat-analysis integration
