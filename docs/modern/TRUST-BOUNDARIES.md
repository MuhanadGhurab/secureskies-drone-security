# Trust Boundaries (Modern View)

> Modern analysis overlay — not a claim that all boundaries were enforced historically.

| Boundary | Notes |
|----------|-------|
| Drone edge | Airframe sensors/effectors concept |
| Flight controller | Pixhawk-class control plane |
| Companion computer | Raspberry Pi-class processing |
| Ground station | Mission Planner / operator tooling |
| Operator | Human-in-the-loop authority |
| Optional cloud | Storage/processing — historically AWS Console concepts only |
| Evidence storage | Future-safe artifact vault for twin outputs |

Commands that physically move an aircraft must never be implied by documentation alone.
