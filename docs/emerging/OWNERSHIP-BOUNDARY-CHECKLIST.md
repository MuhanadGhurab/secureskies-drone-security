# Ownership Boundary Checklist — EMERGING.1

Use before adding files to this public repository.

| Question | Required answer for public commit |
|----------|-----------------------------------|
| Is this the raw academic PDF? | Must be **No** |
| Does it include student IDs or signatures? | Must be **No** |
| Is historical work presented as sole ownership? | Must be **No** |
| Are third-party UI/maps/logos redistributed? | Must be **No** without rights |
| Are failed/inconclusive tests disclosed where relevant? | Must be **Yes** |
| Is Second Place wording including artifact-pending honesty? | Must be **Yes** |
| Is private-source intake linked publicly? | Must be **No** |

If any required answer fails, keep the material private or rewrite as sanitized summary.
