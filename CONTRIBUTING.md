# Contributing

This is a portfolio documentation repository maintained by **Muhanad Ghurab**.

## Before opening a pull request

1. Do not add PDFs, report page images, maps, student IDs, or private extracts
2. Do not invent historical completion claims
3. Keep failed and inconclusive test outcomes visible when discussing testing
4. Keep historical academic content and modern proposals clearly separated
5. Prefer new Mermaid/SVG reconstructions over third-party asset reuse

## Pull requests

- Use clear titles
- Explain why the change improves accuracy, clarity, or safety
- Wait for maintainer review before merging

## Issues

Use issue templates. Do not paste private academic material into issues.
